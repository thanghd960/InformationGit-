Uninstall an extension

If an extension has an icon in your Chrome toolbar, you can right-click the icon and select Remove from Chrome.

If you don't see the extension's icon:

  1. Open Chrome
  2. Click Menu Icon at the top right corner.
  3. Go to More Tools
  4. Go to Extensions.
  5. Next, find the extension you want to remove and click the trash can icon
  6. Click Remove
